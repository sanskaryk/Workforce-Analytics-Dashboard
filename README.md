# Workforce Analytics Dashboard 📊

## 📌 Overview
This project presents an interactive HWorkforce Analytics Dashboard created using Power BI. It provides insights into employee data, attrition trends, and workforce performance to support data-driven decision-making.

<img width="1305" height="737" alt="c" src="https://github.com/user-attachments/assets/131244e3-7915-431a-8947-743dd9351146" />

---

## 🎯 Objectives
- Analyze employee attrition patterns  
- Identify key factors affecting employee retention  
- Visualize workforce distribution and performance  
- Support HR in strategic planning  

---

## 📊 Features
- Attrition analysis (by age, department, role, etc.)
- Employee demographics overview  
- Salary and job satisfaction insights  
- Dynamic filters and interactive visuals  
- KPI indicators for quick insights  

---

## 🛠️ Tools & Technologies
- Power BI  
- Data Visualization  
- Data Cleaning & Transformation  

---

## 📁 Project Files
- `Hr_analytics_dashboards.pbix` → Main Power BI dashboard file  
- `Employee Sample Data.xlsx` → Dataset used for analysis  

---

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open it using Power BI Desktop  
3. Explore the dashboard using filters and visuals  

---

## 📈 Insights Gained
- Key drivers of employee attrition  
- Departments with high turnover  
- Impact of salary and job role on retention
